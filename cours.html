Cours : Bases de la Logique et Algorithmique

1 La Mémoire et les Variables

En programmation, il est crucial de visualiser la mémoire (Heap et Stack). Une variable est un espace alloué dans la RAM dont la valeur peut être modifiée. Une constante (const) est immuable une fois son adresse mémoire assignée.

let score = 0; // Variable stockée dans le Stack
const PI = 3.14159; // Valeur constante

2 Les Structures Itératives (Boucles)

Les boucles automatisent les tâches répétitives. Il est important de gérer les conditions d'arrêt (Edge-Cases) pour éviter les boucles infinies qui saturent le processeur.

  • ▸
    La boucle for : Idéale quand le nombre d'itérations est connu à l'avance.
  • ▸
    La boucle while : S'exécute tant que la pré-condition (Input rule) est vraie.
  • ▸
    La boucle do...while : S'exécute toujours au moins une fois avant d'évaluer la condition.

3 Loi de Pareto (80/20) en Code

"80% des bugs d'un programme proviennent de 20% du code (généralement la logique de base). Pensez toujours État (State) et Action avant la Syntaxe."
